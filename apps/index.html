Getting Started

Fast apps work instantly - no server needed

# For FAST apps (no server needed) - just serve static files:
cd apps
python3 -m http.server 8765
open http://localhost:8765/

# For FULL-TEXT SEARCH across all 6.8M speeches:
python3 build_fts_database.py  # One-time: build FTS5 database (~14GB)
python3 api_server.py           # Run API server for FTS search
# Then select "FTS API (6.8M)" mode in Research Studio
